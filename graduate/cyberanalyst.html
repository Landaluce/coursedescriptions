Cyber Security Analyst / Network Security Administrator

Track Information

General Information

Applicable Graduate Degree(s)

  • MS in Information Technology

Description

This is a common operational security role. Individuals would provide security rules, audit logs, and events, undertake correlation investigations, etc.

Courses to Consider

Required Courses

  • COMP 163: Discrete Structures

  • COMP 170: Introduction to Object-Oriented Programming

  • COMP 264: Introduction to Computer Systems

  • COMP 271: Data Structures I

  • COMP 417: Social, Legal, and Ethical Issues in Computing

  • Three further electives

Recommended Courses

  • COMP 443: Computer Networks

  • COMP 447: Intrusion Detection

  • COMP 448: Network Security

  • COMP 449: Wireless Networks and Security

  • COMP 451: Network Management

  • Database Administration Special Topic
